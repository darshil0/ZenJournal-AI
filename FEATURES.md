# ZenJournal AI - Proposed Features

Based on an analysis of the current ZenJournal AI application, here are three innovative feature ideas to enhance the user experience and deepen the value of AI-assisted journaling.

## 1. Mood-Driven UI Themes (Dynamic Personalization)
Currently, the app has a minimalist, calm aesthetic. To make the journaling experience more immersive, the app could dynamically adjust its theme (accent colors, background gradients, and typography weight) based on the detected mood of the currently selected entry.
- **How it works**: When an entry is tagged as "Joyful," the UI could adopt warm yellow/orange accents. For "Calm," it could shift to soft blues or greens. "Grateful" could feature elegant serif fonts and gold highlights.
- **Value**: Provides immediate visual feedback on the emotional state of a reflection and creates a unique atmosphere for each journal entry.

## 2. AI-Powered Prompt Library (Personalized Inspiration)
While the app currently provides a follow-up prompt after generating insights, a dedicated "Prompt Library" would help users who are stuck before they even start writing.
- **How it works**: A sidebar or modal where users can browse prompts categorized by goal (e.g., "Process Anxiety," "Celebrate Wins," "Find Clarity"). The AI can suggest specific prompts based on the themes detected in the user's *previous* entries.
- **Value**: Reduces the "blank page syndrome" and encourages consistent journaling by providing highly relevant, personalized starting points.

## 3. Mood Trends & Interactive Data Visualization (Insightful Progress)
The Weekly Summary currently provides a text-based distribution of moods. Adding interactive charts would make these insights more actionable and engaging.
- **How it works**: Implement a "Trends" tab within the Weekly Summary modal featuring:
    - **Mood Heatmap**: A calendar view showing mood patterns over the month.
    - **Correlation Chart**: Visualizing the relationship between specific tags (e.g., #work, #exercise) and mood scores.
    - **Theme Evolution**: A streamgraph showing how recurring themes change in prominence over time.
- **Value**: Helps users identify long-term patterns and triggers, transforming a simple journal into a powerful tool for emotional intelligence and self-discovery.
